# ✈️ PrettyFlights — Totem de Check-in

Módulo de autoatendimento para check-in em aeroportos da rede PrettyFlights.

## Versão
`v1.0.0` (em desenvolvimento)

## Tecnologias
- Node.js
- Leitura de QR Code e código de barras
- Interface touchscreen
- Integração com API de reservas PrettyFlights
